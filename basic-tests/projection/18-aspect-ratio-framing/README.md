# 18: aspect-ratio framing

Framing board with four corner markers for comparing wide and tall output aspect ratios.

## Settings
- Default material is fine.
- Resize the window to both wide and tall aspect ratios while keeping the scene centered.

## Expected result
- The center board stays centered and undistorted.
- Corner markers remain anchored to the same scene corners after aspect changes.

## Common failure signs
- The scene stretches non-uniformly when the viewport aspect changes.
- Markers drift or crop unpredictably under resize.
