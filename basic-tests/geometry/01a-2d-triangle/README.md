# 01a: 2D triangle

Smallest possible sanity check for triangle submission, rasterization, and face visibility.

## Settings
- Default material is fine.
- Default camera should show the whole triangle immediately after load.

## Expected result
- A single centered triangle is visible.
- All three vertices are connected with no missing edge or split face.

## Common failure signs
- Nothing renders.
- The triangle appears flipped, clipped, or offset off screen.
