# 03: 3D cube

Basic 3D projection and cube-face rasterization check with a repeated checker texture.

## Settings
- Default material is fine.
- Rotate the camera slightly if you want to inspect multiple cube faces.

## Expected result
- The front, top, and side faces are stable and correctly projected.
- The checker texture stays upright on each face.

## Common failure signs
- Cube edges shear or disappear under camera motion.
- The checker stretches incorrectly on one face.
