# 28: specular highlight shininess

Sphere and floor layout for tuning specular strength and shininess while watching highlight size and motion.

## Settings
- Default material is fine.
- Use the Material Editor to increase shininess and specular strength while orbiting the camera.

## Expected result
- Higher shininess tightens the sphere highlight.
- The highlight moves with the light and camera instead of sticking to the framebuffer.

## Common failure signs
- Changing shininess does not affect highlight size.
- The highlight appears detached from the sphere curvature.
