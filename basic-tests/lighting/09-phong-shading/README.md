# 09: phong shading

Smooth sphere and floor setup for adjusting ambient, specular, and shininess parameters.

## Settings
- Default material is fine.
- Use the Material Editor sliders to change shininess and specular strength.

## Expected result
- The sphere shows diffuse shading and a view-dependent highlight.
- The floor provides a stable textured reference under the same light.

## Common failure signs
- Highlights are fixed in screen space instead of moving with the camera.
- Lighting looks flat regardless of shininess changes.
