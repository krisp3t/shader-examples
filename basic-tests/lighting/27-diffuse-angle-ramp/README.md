# 27: diffuse angle ramp

Five identical cards rotated through different angles to the light for checking Lambert-style diffuse response.

## Settings
- Default material is fine.
- Keep the default light position for the cleanest progression.

## Expected result
- Cards facing the light are brightest, and glancing-angle cards are darker.
- The brightness ramp changes smoothly across the row.

## Common failure signs
- Angle changes have little or no impact on diffuse lighting.
- Cards brighten in the wrong order relative to their normals.
